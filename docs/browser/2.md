### 浏览器渲染

1-5 是在渲染主线程
6-8 合成线程

1. parseHtml

- 生成 dom 树，css 树
- 遇到 js，执行 js

2. 样式计算

- 树合并

3. 布局-Layout
   dom 树与 layout 树大多时候并非一一对应

- 伪元素，display: none
- 内容必须处于行盒中
- 行盒和块盒不能相邻
- 存在匿名行盒，匿名块盒

4. 分层-Layer

- 堆叠上下文
- css will-change 属性

5. 绘制-Paint

- 为每一层生成绘制指令集（指令集合），类似 canvas 绘制

6. 分块-Tiling

- 分块工作是由多个线程执行的

7. 光栅化-Raster

- 光栅化是将每个快变成位图
- 优先处理靠近视口的块

8. 画-Draw

### 面试题

1. 什么是 reflow/回流

- 改变页面结构的 dom 或者样式变化，几何变化
- 异步操作，会将多次操作合并

2. 什么是 repaint/重绘

- 可以跳过 layout 过程
- reflow 一定会导致 repaint

3. 为什么说 transform 效率高

- 在合成线程做的操作，不会阻塞主线程
- 直接执行 draw
